# BatchHash

Calculate  files' hash code(md5)
  - allow wild-card
  - allow recursive search

Usage:
```
$ python BatchHash.py -h
usage: Calculate the check-sum(md5) of files.
BatchHash.py [-h] [-d DIR] [-r] files

positional arguments:
  files              specify file(s).

options:
  -h, --help         show this help message and exit
  -d DIR, --dir DIR  folder to search. current folder if ignored.
  -r, --recursive    perform recursive. default is false
```
